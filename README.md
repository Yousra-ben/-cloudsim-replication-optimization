# Simulation CloudSim avec Optimisation

## Lancer la simulation sur GitHub Actions

1. Allez dans l'onglet **Actions**
2. Cliquez sur **Pipeline CloudSim**
3. Cliquez sur **Run workflow**
4. Attendez la fin (2-5 minutes)
5. Téléchargez les fichiers CSV dans **Artifacts**

## Fichiers générés

- 19 fichiers CSV contenant :
  - État avant drift
  - Impact des drifts
  - État après optimisation
  - Résultats de réplication
